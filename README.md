# qym-headphones
3d prints diy headphones

<img src="https://raw.githubusercontent.com/basson/qym-headphones/main/images/photo_3.jpg" width="400">

## **Ear_Cups.stl**

* Supports: only on holes
*Infill: 25%
*Infill type: Gyroid
*Perimeters: 4/5
*Top solid layers: 0

## **Headband.stl**

* Supports: No 
* Infill: 30%
* Infill type: Gyroid
* Perimeters: 4/5
* Top solid layers: 0
* Bottom solid Layers: 0


## **Rotator_Clips.stl**

* Supports: only on pins
* Infill: 50%
* Infill type: Gyroid
* Perimeters: 4/5
* Top solid layers: 0
* Bottom solid Layers: 0
**Place two nuts on layer 49**

## **Sliders.stl and Driver.stl**

* Supports: No 
* Infill: 20%
* Infill type: Gyroid
* Perimeters: 4/5
